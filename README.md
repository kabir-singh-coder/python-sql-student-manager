# python-sql-student-manager

Student Record Manager using Python and SQL.

This project uses basic Python input statements and SQL queries to manage student records.

Designed as a basic Class 12 IP level project.

## Python

Takes input from the user:
- Roll Number
- Name
- Marks

## SQL

Performs:
- Table Creation
- Insert Record
- Display Records
- Delete Record
- SQL Aggregate Functions

## SQL Functions Used

- MAX()
- MIN()
- AVG()
- COUNT()

## Technologies Used

- Python
- SQL
- SQLite

## Purpose

To understand the basic combination of Python and SQL for storing and managing student data.
